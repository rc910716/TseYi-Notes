# 环境：ubuntu14

## 配置虚拟机和主机之间粘贴复制

1. 方法1，安装open-vm-tools-desktop

```shell
sudo apt-get autoremove open-vm-tools
sudo apt-get install open-vm-tools-desktop
sudo reboot
```

## 虚拟机网络配置

网络相关配置文件为：==/etc/network/interfaces==

```shell
# interfaces(5) file used by ifup(8) and ifdown(8)
auto lo
iface lo inet loopback

#下面为自动获取IP
#auto eth0
#iface eth0 inet dhcp

#下面是设置静态IP
#如果是设置静态IP的话，就需要自己配置网关，DNS域名解析，子网掩码
auto eth0
iface eth0 inet static

        address 192.168.11.137	#静态IP
        netmask 255.255.255.0	#子网掩码
        gateway 192.168.11.2	#网关，最后一位是2,其它位和静态IP的前三位一样
        dns-nameservers 8.8.8.8 8.8.4.4	#网关

```

==重启网络==

```shell
sudo ifdown eth0 && sudo ifup eth0
```

## SSH远程命令行配置

就只是需要下载一些工具而已

```shell
sudo apt-get install openssh-server

#查看是否启动
sudo ps -e | grep ssh
#如果没哟启动，可以执行下面的语句来启动
sudo service ssh start
```
